### Installation
- Laravel dependencies:
  ```sh
  composer install
  cp .env.example .env
  php artisan key:generate
  ```
- React dependencies:
  ```sh
  npm install
  cp .env.example .env
  ```
  
### Contributing

2. Create a new branch for your changes:
